Politique de confidentialité

La protection de vos données personnelles est notre priorité.

Qui nous sommes

L’adresse de notre site Web est : https://pigier-cameroun.com/

Commentaires

Lorsque les visiteurs laissent des commentaires sur le site, nous collectons les données affichées dans le formulaire de commentaires, ainsi que l’adresse IP du visiteur et la chaîne de l’agent utilisateur du navigateur pour faciliter la détection du spam.

Une chaîne anonymisée créée à partir de votre adresse e-mail (appelée « hachage ») peut être fournie au service Gravatar. Politique de confidentialité de Gravatar.

Médias

Si vous téléchargez des images sur le site, évitez celles contenant des données de localisation (GPS EXIF). Les visiteurs peuvent extraire ces données.

Cookies

Si vous laissez un commentaire, vous pouvez enregistrer votre nom, adresse e-mail et site Web dans des cookies pour votre commodité. Ces cookies durent un an.

Lors de votre connexion, nous utilisons aussi des cookies pour mémoriser vos choix d’affichage. Si vous sélectionnez « Se souvenir de moi », votre session dure deux semaines.

Contenu intégré provenant d’autres sites Web

Les articles de ce site peuvent inclure du contenu intégré (vidéos, images, etc.). Ces sites peuvent collecter des données sur vous et suivre vos interactions.

Avec qui partageons-nous vos données

Si vous demandez une réinitialisation de mot de passe, votre adresse IP sera incluse dans l’e-mail de réinitialisation.

Durée de conservation des données

Les commentaires et leurs métadonnées sont conservés indéfiniment pour éviter la modération répétée. Les comptes utilisateurs peuvent modifier ou supprimer leurs informations à tout moment.

Vos droits sur vos données

Vous pouvez demander à recevoir une copie ou la suppression de vos données personnelles. Cela n’inclut pas les données que nous devons conserver pour des raisons administratives, légales ou de sécurité.

Où vos données sont envoyées

Les commentaires des visiteurs peuvent être vérifiés via un service automatisé de détection de spam.
